# Outreach Message Templates

Use these only after finding a specific LinkedIn signal. Always personalize the first line.

---

# Template 1: Job Post Signal

Hi [Name], I saw that [Company] is hiring for [Role].

The job post mentioned [specific responsibility: scheduling / emails / CRM / spreadsheets / customer follow-up]. That usually means the team has repetitive admin work that could possibly be automated.

One quick idea: [specific automation idea].

If useful, I can send you a free 1-page automation opportunity report showing 2-3 workflows that could save time for your team. No pressure.

# Rules

Do:
- Mention the exact LinkedIn signal.
- Keep it short.
- Offer a useful report before selling.
- Make the idea specific.

Do not:
- Send generic AI automation messages.
- Mention university or lack of degree.
- Overpromise results.
- Spam many people with the same text.
