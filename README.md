# 09 LinkedIn Research System

This folder is for finding companies that publicly show business problems on LinkedIn, then converting those signals into automation opportunities.

## Important rule
Use this ethically: do not scrape LinkedIn aggressively, bypass login/paywalls, spam people, or collect private data. Use public posts, job posts, company pages, and manual research.

## Files

1. `LinkedIn_Automation_Opportunity_Finder.html` — interactive HTML tool with offers, filters, scoring, saved companies, outreach message generator, and CSV export.
2. `01_LinkedIn_Filter_System.md` — the exact LinkedIn filters/search process.
3. `02_Pain_Signal_Search_Terms.md` — keywords that reveal automation problems.
4. `03_Company_Research_Sheet_Template.csv` — spreadsheet template with scoring formulas.
5. `04_Scoring_System.md` — how to score opportunities from 0 to 10.
6. `05_Outreach_Message_Templates.md` — messages to contact companies.
7. `06_Automation_Opportunity_Report_Template.md` — 1-page report to send prospects.
8. `07_Daily_Workflow.md` — daily routine to find and contact companies.
9. `08_Search_Query_Bank.csv` — ready-to-use search queries.
10. `09_Google_Sheets_Setup.md` — how to import and configure the spreadsheet.

## Goal
Find companies with score **7+**, send a personalized message, offer a free/cheap automation opportunity report, then sell a small first automation.
